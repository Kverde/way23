# {{ $frontmatter.title }}

Рассмотрим как отредактировать таблицу Oracle через PL/SQL Developer и получить SQL-скрипт изменений.

В SQL-window после ввода схемы и нажатии точки появляется автодополнение со всеми объектами в схеме.

![Автодополнение PL/SQL Developer](images/Edit_tables_1.png)

При клике на полном имени таблицы ПКМ появляется меню с пунктами `View` и `Edit`. Первый пункт позволяет только просматривать, а второй ещё и редактировать таблицу.

![Контекстное меню таблицы](images/Edit_tables_2.png)

После нажатия `Edit` появляется новое окно с вкладками:

- `General` — общая информация о таблице.
- `Columns` — список полей.
- `Keys` — список ключей.
- `Checks` — список check constants.
- `Indexes` — список индексов.
- `Privileges` — список привилегий.

![Форма редактирования таблицы PL/SQL Developer](images/Edit_tables_3.png)

Предположим, нам нужно изменить размерность одного из полей. Заходим на вкладку `Columns`.

![Редактирование колонок таблицы](images/Edit_tables_4.png)

В колонке `Type` меняем размерность нужного поля

![Изменение размерности поля](images/Edit_tables_5.png)

Теперь есть несколько вариантов. Можно сразу применить изменения кнопкой `Apply` или можно получить скрипт нажав кнопку `View SQL`.

![Применение изменений таблицы](images/Edit_tables_6-e1564461593897.png)

Попробуем получить скрипт.![Просмотр скрипта на изменение](images/Edit_tables_7-e1564461689666.png)

Копируем его и отменяем изменения в таблице кнопкой `Refresh`.

![Отмена изменений таблицы](images/Edit_tables_8-e1564461704492.png)

Теперь в редакторе таблицы нет изменений. При нажатии на `View SQL` будет показан полный скрипт создания таблицы.

![Просмотр полного кода таблицы](images/Edit_tables_9-e1564461722361.png)
